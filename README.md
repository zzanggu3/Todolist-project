# Todolist 과제 (Next.js + TypeScript)

## 프로젝트 개요
Next.js와 TypeScript를 활용하여 만든 간단한 Todolist 웹 애플리케이션입니다.  
- 할 일 추가, 완료/미완료 토글 기능  
- 완료한 항목은 DONE 리스트로 이동  
- 웹, 태블릿, 모바일 반응형 UI 지원 (기본적인 스크롤 대응)

## 기술 스택
- Next.js 13 (App Router)
- TypeScript
- React
- CSS-in-JS (inline 스타일)
